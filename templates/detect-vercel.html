⚠️ Vercel Deployment Notice

This is a UI demonstration deployed on Vercel's serverless platform.

Due to Vercel's serverless architecture limitations, webcam streaming and real-time emotion detection are not available.

For full functionality with live emotion detection, please use one of these alternatives:

🚀 Deploy to Render 🚂 Deploy to Railway 💻 Run Locally

📋 Why This Limitation Exists

Vercel's serverless functions have constraints:

  • ❌ Cannot access user's webcam directly
  • ❌ 500 MB deployment size limit (TensorFlow alone is ~800 MB)
  • ❌ 10-60 second execution timeout
  • ❌ No persistent connections for video streaming

🚀 How to Run Locally (Full Features)

  1. Clone the repository
  2. Install dependencies: pip install -r requirements-full.txt
  3. Run the app: python app.py
  4. Open: http://localhost:5000
  5. Allow camera permissions when prompted
  6. Enjoy real-time emotion detection! 🎉

☁️ How to Deploy to Render (Recommended)

  1. Push your code to GitHub
  2. Go to render.com
  3. Click "New +" → "Web Service"
  4. Connect your GitHub repository
  5. Use build command: pip install -r requirements-full.txt
  6. Use start command: gunicorn wsgi:app
  7. Add environment variables (SECRET_KEY, FLASK_ENV)
  8. Deploy and enjoy full functionality! 🎉
